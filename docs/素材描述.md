# 素材描述文档

本文档详细描述了项目中使用的所有素材文件，包括3D建模作品和界面设计作品。

## 3D建模作品素材

### 1. 化妆品建模
- **路径**: `assets/3D Modeling Page/01化妆品.jpg`
- **描述**: 这是一个精细的化妆品3D建模作品，展现了化妆品瓶身和包装的细节设计。模型展现了光滑的材质质感，包括玻璃瓶身、金属盖子等不同材质的渲染效果，整体呈现出高端化妆品的精致感。

### 3. 家具建模

### 5. 石膏像建模
